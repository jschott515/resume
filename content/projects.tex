\section{Projects}

\textbf{PyCIE - Cross-Platform Python XDMA Driver} \\

\begin{itemize}
    \item Leverged the Xilinx XDMA Windows and Linux kernel drivers to create a portable Python driver for memory access to FPGAs via PCIE.
    \item Created a platform-independent interface for DMA and bypass channel reads/writes using device driver files.
    \item Implemented Windows support with ctypes wrappers for the Window's file API; Linux support with OS and MMap.
\end{itemize}
